# 🤖 Telegram Bot для обучения C# 

Обучающий Telegram-бот для изучения основ языка программирования C#. Проект предлагает интерактивное обучение, совмещающее теоретические материалы и практические тесты с системой уровней и баллов.

## 🌟 Основной функционал

*   **📖 Изучение теории:** Текстовые уроки с поддержкой HTML-форматирования Telegram.
*   **🧠 Интерактивное тестирование:** Тесты с вариантами ответов (Inline-кнопки). За каждый правильный ответ начисляются баллы (по умолчанию 10 баллов).
*   **📈 Система прогрессии:** Переход на следующий уровень сложности происходит автоматически при достижении заданного порога баллов , который настраивается индивидуально для каждого уровня.
*   **🛠 Панель администратора:** Встроенная админка на базе FSM (Машины состояний) для добавления, редактирования и удаления вопросов прямо через интерфейс бота, без необходимости лезть в код базы данных.
*   **💾 Хранение данных:**  Aрхитектура с использованием JSON для хранения контента и прогресса пользователей.

## 🛠 Технологии и библиотеки

*   **Язык:** Python 3.11.9
*   **Фреймворк:** [aiogram 3.x](https://docs.aiogram.dev/) (Асинхронный фреймворк для Telegram API)
*   **Архитектура:** FSM (Finite State Machine), Routers
*   **Для защиты секретных данных:** `python-dotenv`

## 🚀 Быстрый старт

### Требования
- Python 3.8+
- Telegram Bot Token (от [@BotFather](https://t.me/botfather))

### Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/YOUR_USERNAME/tg_bot_v2.git
cd tg_bot_v2
```

2. Создайте виртуальное окружение:
```bash
python -m venv venv
# Для Windows:
venv\Scripts\activate
# Для macOS/Linux:
source venv/bin/activate
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

5. Отредактируйте `.env` и добавьте ваши данные:
```
BOT_TOKEN=ваш_токен_бота
ADMIN_IDS=ваш_telegram_id
```

6. Запустите бота:
```bash
python main.py
```
