ЧТО ТАКОЕ ПЕРЕМЕННАЯ? Переменная — это контейнер для хранения данных. 📦 Представь коробку: • имя коробки — это имя переменной • содержимое — это значение 
int a = 5;
💡 Здесь: • a — имя переменной • 5 — значение ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 🔹 2. ОСНОВНЫЕ ТИПЫ ДАННЫХ В C# есть разные типы данных: • int — целое число (например: 5, 10, -3) • double — дробное число (например: 3.14) • string — текст (например: "Hello") • bool — логическое значение (true или false) 
int age = 18;
double price = 9.99;
string name = "Ivan";
bool isStudent = true;
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 🔹 3. ОБЪЯВЛЕНИЕ ПЕРЕМЕННЫХ Чтобы создать переменную, нужно указать тип и имя: 
int a;
string text;
Или сразу задать значение: 
int a = 5;
string name = "Ivan";
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 🔹 4. СТРОКИ (STRING) Строки всегда записываются в двойных кавычках: 
string text = "Hello";
⚠️ Одинарные кавычки не используются для строк! ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 🔹 5. МОЖНО ЛИ МЕНЯТЬ ПЕРЕМЕННЫЕ? Да, значение переменной можно менять: 
int a = 5;
a = 10;
💡 Запомни: переменная — это не константа, её значение можно изменить. ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ ✅ ГОТОВ ПРОВЕРИТЬ СЕБЯ? Нажми «Пройти тест» и ответь на вопросы! 💰 За каждый правильный ответ ты получишь 10 очков Удачи! 🚀
