📚 ЗА ЭТОТ УРОВЕНЬ ТЫ ИЗУЧИШЬ: • 🔁 Что такое циклы • 🔢 Как работает цикл for • ⏳ Как работает цикл while • ⛔ Как остановить цикл • ⚡ Как управлять повторениями ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 🔹 1. ЧТО ТАКОЕ ЦИКЛ? Цикл — это способ повторять один и тот же код несколько раз. 📌 Пример из жизни: • Написать 10 раз одно и то же — долго • Использовать цикл — быстро ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 🔹 2. ЦИКЛ FOR Цикл for используется, когда известно количество повторений. 
for (int i = 0; i < 5; i++)
{
    Console.WriteLine(i);
}
📌 Здесь: • i = 0 — старт • i < 5 — условие • i++ — увеличение 💡 Выведет числа от 0 до 4. ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 🔹 3. ЦИКЛ WHILE Цикл while работает, пока условие истинно. 
int i = 0;

while (i < 5)
{
    Console.WriteLine(i);
    i++;
}
💡 Работает, пока условие true. ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 🔹 4. ОСТАНОВКА ЦИКЛА • break — полностью останавливает цикл • continue — пропускает текущую итерацию 
for (int i = 0; i < 5; i++)
{
    if (i == 3)
        break;
}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 🔹 5. ЗАЧЕМ НУЖНЫ ЦИКЛЫ? Циклы используются для: • перебора данных • повторения действий • обработки массивов ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 🧠 КРАТКО: • цикл = повторение • for — когда знаем количество • while — пока условие true • break — остановка • continue — пропуск ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ ✅ ГОТОВ ПРОВЕРИТЬ СЕБЯ?
